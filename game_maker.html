GAME MAKER

O projeto foi criado pelos alunos Kelvin e Matheus, trata se de uma tentativa de criar um jogo, usando a plataforma Game Maker, foram feitos 2 projetos um foi criar um jogo de Plataforma, e o outro um Dungeon Crawler, Ambos um projeto com um curto prazo, Segue abaixo a explicação dos temas:



O jogo de plataforma criado por Matheus segue um padrão de movimento de 2 direções esquerda e direita, neste jogo seu objetivo é passar as fazes coletando o maior numero de moedas com menos mortes, desviando dos inimigos





O jogo criado por Kelvin segue um movimento de 8 direções, cima, baixo, direita, esquerda, e as quatro diagonais, o objetivo do jogo é derrotar todos os inimigos de uma sala para progredir para próxima sala, e mudando de andares, e acrescentando na dificuldade.
